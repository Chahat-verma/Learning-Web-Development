Seo Practices

Title used to give title to the webpage

Meta tags are used to give information about the webpage

Meta description is used to give a brief description of the webpage

Meta keywords are used to give keywords related to the webpage

Meta robots are used to give instructions to the search engines

Meta viewport is used to control the layout on mobile browsers

Meta charset is used to specify the character set used in the webpage

Meta author is used to specify the author of the webpage

Meta copyright is used to specify the copyright of the webpage

Meta refresh is used to refresh the webpage after a certain time

Meta http-equiv is used to specify the HTTP headers of the webpage

Meta content is used to specify the content of the webpage

Meta name is used to specify the name of the webpage
